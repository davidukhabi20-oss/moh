# Velora Tomato

Velora Tomato is a premium web project showcasing the "Gold Standard of Tomato Richness". The project provides an elegant interface for customers to learn about the brand, view products, and place orders for high-quality tomato paste.

## 📁 Project Structure

The project follows a clean, deployment-ready structure:

```
.
├── index.html            # Home page
├── about.html            # Brand story and values
├── product.html          # Detailed product information
├── order.html            # Product range and ordering
├── our team.html         # Team profiles
├── testimonials.html     # Customer feedback
├── contact.html          # Contact information
├── reviews.html          # Customer review gallery
├── css/
│   └── style.css         # Global styles and layout
├── js/
│   └── main.js           # Client-side interactivity
└── images/               # All project assets (logos, products, team photos)
```

## 🚀 How to Run

Since this is a static web project, you can run it in several ways:

### 1. Local File Opening
Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

### 2. Using VS Code Live Server
1. Install the **Live Server** extension in VS Code.
2. Right-click `index.html` and select **"Open with Live Server"**.

### 3. Simple Python Server
If you have Python installed, run the following command in the project root:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
